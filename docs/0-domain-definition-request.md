# 너는 최고의 비즈니스 도메인 전문가야. "JKN-TODOLIST"라는 이름의 사용자 기반의 인증과 함께 사용하는 ToDoList 애플리케이션을 개발하려 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘

## 문제 정의

- 사용자마다 할일 목록을 관리할 수 있는 앱이 좋겠다.
- 할일의 만료 기간이 표현되어 할일 목록 관리할 수 있으면 좋겠다.
- 국경일 같은 일정(할일)이 함께 관리될 수 있어야 한다.
- 향후 확장 가능 기능에 대해 고려해야한다.

## JKN-TODOLIST 앱의 주요 기능

- 회원 가입을 한 다음 로그인 한 사용자만 앱의 주요 기능을 이용할 수 있다.
- 로그인 한 사용자는 자신만의 할일 목록을 조회할 수 있다.
- 할일 목록에서는 새로운 할일 추가, 기존 할일 완료, 할일 수정 및 삭제가 가능해야한다.
- 할일은 일정(시작, 만료시기)과 할일의 내용을 포함한다.
- 기존 할일을 삭제한 후 삭제한 할일은 휴지통으로 돌아가고 다시 원상복구할 수 있다.
- 휴지통에서 삭제된 일정은 되돌릴 수 없는 영구삭제를 한다.
- 국경일과 같은 공통 일정은 삭제할 수 없다.
- 기술스택은 BE: NODE + EXPRESS / 4. FE: React 19~18 / 로그인 인증 토큰은 JWT 사용하고자하는데 이단계에서 고민하는 내용이아니면, 차후에 다시 구체화하자

## 향후 확장 기능

- 현재 접속한 사용자의 브라우저 정보와 접속 ip 등을 기록하는 부가 기능
- 현재 접속한 사용자의 현재 위치를 가져올 수 있도록 qr이미지를 만들고 그 이미지를 사용자의 휴대폰단말로 인증하면 현재 위치가 가져와지도록 하는 부가기능
