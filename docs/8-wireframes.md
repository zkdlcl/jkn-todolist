# 8. 와이어프레임 (Wireframes)

## 8.1 로그인/회원가입 페이지

### 레이아웃

- 중앙 정렬 폼 (최대 너비 400px)
- 로고 상단 중앙 배치

│  │ 이메일                     │  │
│  └───────────────────────────┘  │
│  ┌───────────────────────────┐  │
│  │ 비밀번호                   │  │
│  └───────────────────────────┘  │
│  ┌───────────────────────────┐  │
│  │      로그인 (파랑)         │  │

## 8.6 색상 및 스타일 가이드

### 색상

- Primary: #3B82F6 (파랑)
- Success: #10B981 (초록)
- Warning: #F59E0B (주황)
- Danger: #EF4444 (빨강)
- Gray: #6B7280
- Background: #F9FAFB

### 타이포그래피

- H1: 24px (페이지 제목)
- H2: 20px (섹션 제목)
- Body: 14px (본문)
- Small: 12px (보조 텍스트)
- Font: 한국사람 사용 APP으로 JEJUMYEONGJO.TTF 위주 사용 제목과 메인타이틀은 JEJUHALLASAN.TTF 사용

### 컴포넌트 상태

- 기본: 테두리 #E5E7EB
- Hover: 배경 #F3F4F6
- Active: 테두리 #3B82F6
- Disabled: 투명도 50%
